EAGLE Resources

I am an avid user of Autodesk EAGLE and often create libraries, scripts, and design rules to improve my workflow.

One such script takes the board file and automatically generates an Excel file for each drill diameter containing the locations of the holes. In addition, it generates eight versions of transformations to properly align the holes when imported into Autodesk Inventor for mechanical CAD work.
